# Quality Non-Conformance Analytics Dashboard made with Power BI

## Overview
  This project analyzes manufacturing non-conformance reports (NCRs) to support root cause analysis, corrective and preventive actions, and quality KPI tracking within a Quality Management System (QMS).

## Objectives
  Analyze NCR trends by department and time

  Support 8D root cause analysis and corrective action tracking

  Develop actionable quality KPIs (severity, closure rate, customer impact)

  Identify supplier and process-related quality risks

## Tools & Technologies
  SQL (data storage, querying, aggregation)

  Power BI (dashboard development and visualization)

  Excel / CSV (data generation and preprocessing)

## Key Dashboards
  NCR Trends by Department and Month

  Root Cause & 8D Status Analysis

  Quality KPIs (Severity, Closure Rate, Customer Impact)

## Skills Demonstrated
  NCR analysis

  Root Cause Analysis

  Quality KPI definition and tracking

  SQL-based data modeling

  Power BI dashboard design

  Data-driven decision support in QMS environments
